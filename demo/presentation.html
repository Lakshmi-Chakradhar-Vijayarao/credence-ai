① Haiku — Compression Agent
Fast. Cheap. Blind to epistemic weight.
Compresses resolved context at <$0.001/turn. Cannot distinguish "I think 50" from "50" — identical token distribution. Qualifier is collateral loss. Probe fires before Haiku ever sees uncertain text.
$0.80/$4 per 1M tok · called only on HIGH-J turns
skipped when uncertainty markers present in user turns
② Opus 4.7 — Reasoning Agent
The ghost detector is why it must be Opus 4.7.
Ghost Detector — no hedging markers, sounds certain. Haiku returns []. Opus 4.7 reasons about provenance: where each claim came from, not just what it says. Catches the case every other layer misses.

Also: main generation · contradiction scan · adaptive thinking in uncertain territory. None of this is replaceable with a smaller model.
ghost detect: ~$0.002/call · n=10 gauntlet · 100% detection
Haiku misclassifies ghost constraints · Opus catches all 10
